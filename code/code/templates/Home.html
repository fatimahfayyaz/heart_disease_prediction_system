Heart Disease Detection System

Project Aim

The main purpose of this project is to predict whether a person has heart disease or not?


Input and output

Input: Person's age, gender, slope, fbs VALUE

Output: Prediction (Detected / Not Detected)


Representation of Input (PERSON'S VALUES OF Age, Gender, SLOPE, FBS)

  Input (PERSON'S VALUES) is Represented as 4 Input Attributes

  Input Attributes 01– age

    Possible Value 01 – 1 (20-30)

    Possible Value 02 – 2 (30-40)

    Possible Value 03 – 3 (40-50)

    Possible Value 04 – 4 (50-60)


  Input Attributes 02– gender

    Possible Value 01 – Male

    Possible Value 02 – Female



  Input Attributes 03– slope

    Possible Value 01 – 0

    Possible Value 02 – 1

    Possible Value 03 – 2


  Input Attributes 04– fbs

    Possible Value 01 – 0

    Possible Value 02 – 1


Representation of Output

   Output is Represented as a single Output Attribute

    Output Attribute – Detection

    Possible Value 01 – Detected

    Possible Value 02 – Not Detected


Example - Input and Output

Input (age):

  age:

    3

  gender:

    3

  slope

  fbs:

    2
